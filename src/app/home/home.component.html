This is a service to upload and view GraphQL Schemas. This was built with the idea that GraphQL will be available as a third-party api. This service will make it simple for people to document and share Schemas.

What was this built with?

GONG Stack

Mongo -> GoLang (Echo) -> Rxjs -> Typescript -> Angular (Material)
